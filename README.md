Benchee Jiffy Benchmark

This is clone of the Jason bench https://github.com/michalmuskala/jason with an
extra data set of large numbers and just running jiffy vs OTP 27+ json comparisons.

It's designed to run from the jiffy directory and set up to compare it against
OTP 27+ Erlang/OTP json module. To use clone this to jiffy directory and run:

``` shell
./run_compare.sh <branch=master>
```
